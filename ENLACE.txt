INSTRUCCIONES DE ARRANQUE:
1. Base de Datos: Importar "inventario.sql" en MySQL (Puerto 3306).
2. Backend: 
   - Abrir terminal en carpeta /backend
   - Ejecutar: npm install (solo la primera vez)
   - Ejecutar: node server.js
3. Frontend:
   - Abrir terminal en carpeta /frontend
   - Ejecutar: npm install (solo la primera vez)
   - Ejecutar: npm start
4. Acceso: http://localhost:3000
